Use the left and right arrow keys to move the slug to fall through the rising elevators. Each time the slug falls through an elevator gap it scores a point. If the slug hits any of the red spikes it will be impaled and will lose a life. The game ends when the slug has lost all of its extra lives. Refresh the page to restart the game. You can select extra lives of the slug by marking the checkboxes, which takes effect when a new game is started. The score and number of extra lives are displayed in hexadecimal.
